5         提测

首先来看，没有创建项目的情况下提测，假定你已经登陆.

在你还不是任何项目成员的情况下，你将看不到任何项目也不能提测。这时候你有两个选择。1 在首页查看项目信息找到负责人，把你添加到项目里，然后提测。2 选择自己创建项目，并提测。

进入系统，如果你能看到你想提测的项目，那么请你直接跳到本文的最后面，绿色背景的地方，开始直接创建提测。

我们直接看自己选择创建项目并提测的情况。

3)         创建项目

创建项目的页面如下：

 

          填写以上字段，并点击创建。创建成功在桌面右下角会有绿色弹出框提示项目创建成功。创建完成后，页面上不会立即显示项目，需要你刷新一下页面（此处你可能想骂人，这一版就这样吧，先忍一忍）

       创建完了，项目你是不是觉得就可以开始创建提测了。到这还不可以，需要你对项目做一些设置。创建完项目后，在首页刷新，就可以看到如下页面：

 

 

点击页面中红框里的项目，进入到项目设置页面，如下图

 

点击【设置】，进入到项目设置页面，在以下页面你可以简单修改一下项目信息

 

在设置页面上，点击【成员】tab可以为本项目添加成员。成员添加页面如下：

 

添加完成员，接下来就需要你添加一个用于打包的jenkins job地址。点击【WebHook】Tab看到如下页面

 

URL: URL中输入用于打包的jenkins job的地址

标签：你所添加的URL的标示，别名。（一个英文的字符串）

 

填写完以上信息，点击添加webhook. 看到如下页面，到这里就完成了对项目的配置。如果有兴趣你可以点击【构建】按钮，测试一下你添加的构建URL是否ok

 

接下来，就可以开始创建提测了：

在header上点击【+】，选择提测》功能，会弹出如下的对话框

在根据要求填写各项信息，点击创建，右下角会弹出创建成功提示框。之后依旧需要你直接刷新页面来查看提测。提测创建完成后，你会看到如下的提测条目。

创建完成后，提测条目还处于为提测状态。

如果你认为现在已经可以提测了，那么久点击提测按钮就可以了。
